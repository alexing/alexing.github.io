# alexing.github.io
A portfolio starter for [Gatsby](https://www.gatsbyjs.org/) with a 2 column layout.  

## Demo
See it in action [here](http://www.alexingberg.com/).

## Installation
```
gatsby new gatsby-site https://github.com/alexing/alexing.github.io
```
